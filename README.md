# Simple Calculator (Tkinter)

A simple calculator made with Python and Tkinter.  
It supports **dark and light themes** that you can switch using the theme button.

## Features
- Basic arithmetic: +, -, ×, ÷ (one operation at a time)
- Clear (AC), backspace (↵), and percentage (%) buttons
- Dark/Light mode toggle with icons
- Realistic calculator layout

## Colors

**Dark Mode**
- Background: #1e1e1e
- Buttons: #333333
- Numbers: #f5f5f5
- Clear/Delete: #ff1a1a
- Result: #ffe4c4

**Light Mode**
- Background: #f5f5f5
- Buttons: #dedeff
- Numbers: #000000
- Clear/Delete: #ff0000
- Result: #4d2600
